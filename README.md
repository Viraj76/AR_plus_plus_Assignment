# Tech stack used
* Kotlin
* Jetpack Compose 
* Clean architecture with MVVM
* Coroutines & Flow
* Dagger Hilt
* Paging 3
* RoomDB
* Preference DataStore
* Retrofit
* Jetpack Navigation
